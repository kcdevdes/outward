# Outward

A 3D audio visualizer built with Three.js that creates an interactive music experience with real-time spectrum analysis.

## Features

- Real-time audio spectrum visualization on a 3D cube
- Interactive camera controls (mouse drag to rotate, scroll to zoom)
- Dynamic particle system that responds to audio frequencies
- Supports custom audio file upload
- Includes sample audio playback
- Skybox environment with custom texture

## Technologies

- Three.js for 3D rendering
- Web Audio API for audio analysis
- HTML5 Canvas for spectrum texture generation

## Usage

1. Open the application in a web browser
2. Upload your own audio file or play the sample track
3. Interact with the 3D scene using mouse controls:
   - Drag to rotate camera
   - Scroll to zoom in/out
